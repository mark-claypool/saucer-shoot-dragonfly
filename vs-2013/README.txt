README

This directory contains Microsoft Visual Studio v12 (2013) solution
files.

The Solution assumes a directory setup as follows:

Projects/     Location of Visual Studio projects
--game/       Source code for game
----game/     Visual Studio solution files
------Debug/  Location of .exe that run in Debug mode
--SFML-2.3/   SFML directory
--dragonfly/  Location of Dragonfly include/ and lib/ sub-directories
